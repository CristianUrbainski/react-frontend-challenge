### Instalação

É preciso [Node.js](https://nodejs.org/) v12+ para executar o projeto.

Execute os seguintes comandos na pasta raiz do repositório para executar o projeto:

```sh
$ cd server
$ npm install
$ npm start

$ cd ..\client
$ npm install
$ npm run start
```
